# mcp-scanner
 MCP json scanner that keeps you safe when installing MCPs 


## What it does 
 Takes mcp input as JSON and checks for common hostile flags or permissions. 

## Why
 MCP servers are relatively new, everyone wants them, and hostile MCP servers are a real thing, one misconfig -
 and someone has full access to your whole system. Keep everyone safe and make sure you trust your MCP servers
